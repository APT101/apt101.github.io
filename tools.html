Welcome to the Tools section of APTT. Here you can find tools to help you or your IT Department protect yourself and others. You will find links to both internal and APTT approved tools that were selected to better help in defending yourself against savvy attackers online. The tools include well respected companies, open-source projects, and other companies dedicated to stopping phishing. Be warned, most of these tools require you to engage with the Phishing email, if you do not feel safe doing so, do not worry and just delete the email or tell your IT department.

Phish.Report Link Reporter

Phish.Report allows you to copy phishing links, have them scanned, and uploaded to databases to protect others.

Phish.Report Link Reporter

Have I Been Pwned Breach Checker

This website will tell you if your information has been stolen from an information breach found online.

Have I Been Pwned Official Website

VirusTotal Attachment Checker

This tool allows you to take any file and upload it to this site to have it verify if it is malicious or not.

VirusTotal Attachment Checker

VirusTotal Link Checker

This tool allows you to copy any link and paste it into this site to have it go there and check to see if it is secure or not.

VirusTotal Link Checker

GoPhish Simulated Phishing Attacks

A free open source tool for IT Departments to test you with real looking fake phishing attacks similar to APTT

GoPhish Official Website

MXToolbox Email Analyzer

This tool allows an experienced user to check email headers for spoofing and authentication failures to verify if it is safe or not.

MXToolbox Header Analyzer
